Sample Website UI Test
======================

Requirement
-----------
1. Java JDK
2. Maven
3. Docker

Quick Start
-----------

1. Run [Zalenium](https://github.com/zalando/zalenium)

	```bash
	curl -sSL https://raw.githubusercontent.com/dosel/t/i/p | bash -s start
	```

2. Run test

	```bash
	mvn test
	```
